# DAAN KGP

<br/>

> **Dakshana Alumni Network at IIT Kharagpur**
> A comprehensive ecosystem serving as a bridge between batches, a repository of institutional memory, and a structured support system for the community.

<br/>

## Project Overview

DAAN KGP was envisioned to function not merely as a directory of individuals, but as a unified digital ecosystem built on shared values, mutual support, and collective growth. Prior to its inception, the community relied on fragmented networks, leading to disorganized information, navigation challenges for newly admitted students, and a lack of centralized academic memory.

The DAAN KGP web platform resolves this by offering a robust, scalable, and privacy-first full-stack application tailored to automate administrative coordination, centralize community resources, and visually map the demographic footprint of the organization across the IIT Kharagpur campus.

<br/>

## Core Platform Modules

### Authentication & Security
Features a custom authentication layer ensuring that sensitive personal information, directories, and private dashboards are accessible exclusively to verified DAAN KGP personnel.

### Subsystems & Interfaces

* **Home Dashboard**
  Features dynamic notice boards for upcoming events, real-time organizational growth statistics, an animated CR carousel, and campus exploration tools with map integration.

* **Our-Fam Directory**
  Provides visual analytics indicating gender, batch, branch, hall, and COE distributions. Designed to facilitate secure access to detailed profile cards of all community members.

* **Resource Toolkit**
  A structured, curated knowledge base addressing critical transition phases such as ERP management, freshman campus life, academics, and CDC/internship preparation.

* **Secure Forms Subsystem**
  An authenticated response collection system engineered for surveys, event registrations, and logistical coordination directly linked to the central database.

* **Academic Stars**
  A merit-based public record showcasing top academic performers across batches to foster motivation and recognition.

* **Personal Workspace**
  A private user dashboard incorporating productivity tools (to-do lists, utilities), internal analytics, a secure diary feature for journaling, and dynamic profile management.

## Local Development Workflow

Follow these instructions to mirror the production build locally for development or contribution.

### Prerequisites

Ensure you have Node.js (version 18 or higher) and npm installed on your local machine.

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/daankgptech/daankgp-client
   ```

2. **Navigate to the client directory**
   ```bash
   cd daankgp-client
   ```

3. **Install project dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

Upon running the start command, the development environment will initialize and render on your local port (typically `localhost:5173`).
